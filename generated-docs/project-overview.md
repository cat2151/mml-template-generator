# Project Overview

## プロジェクト概要
- 「mml-template-generator」は、ユーザーが簡単な音符（cdeなど）を入力すると、様々なMMLコンパイラで即座に演奏可能なMMLコードを生成するWebツールです。
- このツールは、MML初心者がコンパイルエラーに直面することなく、手軽にMMLで音楽を試せるよう設計されています。
- シンプルなインターフェースを通じて、最小限の入力からMMLテンプレートを効率的に作成し、MML学習の初期段階での障壁を低減します。

- 音楽・オーディオ: MML (Music Macro Language - 音楽を記述するためのテキストベースの言語。本プロジェクトの核となる出力形式)。

## 関数詳細説明
- **controller (controller.js)**: ユーザーインターフェースのイベント（入力値の変更、選択肢の変更など）を監視し、それに応じてMML生成関数を呼び出すなど、アプリケーションの流れを制御する役割を担います。
- **function (controller.js)**: `controller.js`内で使用される匿名関数、またはプライベートなヘルパー関数である可能性があります。特定のイベントハンドリングやロジックの一部として機能します。
- **getMiddleText (services.js)**: ユーザーが入力した音符列などのテキストを、MMLの生成に適した中間形式に変換する役割を持つ関数です。
- **sion (services.js)**: SionフォーマットのMMLを生成するためのロジックの一部、またはそのフォーマットに特化したMML変換処理を行う関数です。
- **sutoton (services.js)**: SutotonフォーマットのMMLを生成するためのロジックの一部、またはそのフォーマットに特化したMML変換処理を行う関数です。
- **auto (services.js)**: MMLを自動的に生成する処理、またはデフォルト設定や簡略化されたルールに基づいてMMLを生成する機能に関連する関数です。
- **generate (services.js)**: ユーザー入力と選択されたMMLフォーマットに基づき、最終的なMMLコードを生成する主要な関数です。他のMML生成ヘルパー関数を内部で呼び出します。
- **genHead (services.js)**: 生成されるMMLコードのヘッダー部分（例: チャンネル設定、テンポ、音量などの初期情報）を生成する関数です。
- **gen1ch (services.js)**: 単一チャンネルのMMLコード（例: 音符、休符、音長、オクターブなどのシーケンス）を生成する関数です。
- **genTail (services.js)**: 生成されるMMLコードのフッター部分（例: 終了記号、ループ情報など）を生成する関数です。
- **getABC (services.js)**: 音名（C, D, Eなど）の処理や、特定の音名体系への変換、あるいは音符の属性（シャープ、フラットなど）の調整を行う関数です。
- **service (services.js)**: `services.js`内に存在する汎用的なサービス関数、またはアプリケーションのロジックの一部を提供するヘルパー関数です。
- **forEach (services.js)**: JavaScriptの組み込みメソッドである`forEach`、またはそれに類似する反復処理を行うヘルパー関数です。リストや配列の各要素に対して指定された処理を実行します。
- **if (services.js)**: これはJavaScriptの条件分岐構文であり、特定の条件下でコードブロックを実行するために使用されます。関数呼び出し階層では、その条件分岐によって始まる特定の処理フローを示唆しています。
- **bgm (services.js)**: 背景音楽（BGM）に関連するMMLの生成ロジック、またはBGMトラックのMMLを処理する機能に関連する関数です。
